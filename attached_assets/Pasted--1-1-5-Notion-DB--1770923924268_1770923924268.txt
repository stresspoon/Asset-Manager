### 2. 상담 신청 DB — 신규 필드 7개 추가 (노란색 행)

명세서 2.1 테이블에서 `color="yellow_bg"` 행이 모두 새로 추가된 필드입니다.

| Notion 필드명 | 영문 코드 | 타입 | 옵션값 |
| --- | --- | --- | --- |
| `카드 사용 건수 (월)` | `card_usage_count` | Select | `50건 이하` · `51~150건` · `151~400건` · `401건 이상` |
| `카드 개수` | `card_count` | Number | 사용 중인 법인/사업용 카드 수 |
| `세금계산서 발행 건수 (월)` | `tax_invoice_count` | Select | `5건 이하` · `6~20건` · `21~60건` · `61건 이상` |
| `연매출 규모` | `annual_revenue` | Select | `10억 이하` · `10억~50억` · `50억~100억` · `100억 이상` |
| `폼 작성 완료 여부` | `form_completed` | Checkbox | 폼 제출 시 자동 `true` |
| `예약 완료 여부` | `reservation_completed` | Checkbox | 캘린더 예약 시 자동 `true` |
| `접수 완료 여부` | `submission_complete` | Formula | `form_completed AND reservation_completed` |

> **기존 데이터 처리 완료**: 기존 3건 모두 `폼 작성 완료 여부` = ✅, `예약 완료 여부` = ✅ 설정됨

| 서비스 등급 | 등급 코드 | 월 기본료 | 매칭 조건 | 대상 연매출 |

### 6. 개발 영향 범위 요약

- **API 수정 필요**: `/api/requests` 응답에 신규 7개 필드 포함
- **API 수정 필요**: `/api/schedules` 응답에 신규 2개 필드 포함
- **신규 API 필요**: `/api/pricing` (요금 기준표 조회), `/api/quotes/*` (견적서 CRUD + 자동 산출)
- **UI 추가 필요**: 견적 자동 산출 패널 (`/requests/:id`), 견적서 목록/상세 (`/quotes`, `/quotes/:id`)
- **접수 완료 로직**: `submission_complete` Formula가 Notion에서 자동 계산되므로, 웹앱에서는 해당 값을 읽기만 하면 됩니다
